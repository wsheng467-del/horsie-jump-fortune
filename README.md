# 跳一跳「马跃福至」 — 轻盈竞技迎新春

简要介绍
- 《马跃福至》是一款轻松上手的春节主题跳跃竞技游戏。玩家通过手机轻点控制大屏上的小马跳跃，跨越铜钱、福字、春幡等障碍，积累分数并争取高分通关，最终可触发烟花特效庆祝胜利。

核心玩法
- 控制方式：手机轻点（或长按以蓄力，具体视实现而定）。
- 跳跃目标：跨越铜钱、福字、春幡等障碍物或平台。
- 计分规则：跳跃距离越长分数越高；连跳或落在特殊区域可获得连击/加分奖励。
- 关卡要素：障碍布局、移动平台、时间/风向等可变化的环境元素。
- 通关奖励：完成一定距离/关卡后触发烟花特效与分数榜展示。

特色和亮点
- 春节主题美术（福字、春幡、铜钱、烟花等）。
- 简单易上手的单手操作，适合社交传播与短时竞技。
- 可扩展的道具与皮肤系统（例如新年皮肤、加速道具）。
- 排行榜与成就系统，支持朋友间比拼高分。

示例玩法引导（建议）
1. 点击/短按：小跳
2. 长按/蓄力：更远的跳跃
3. 在空中可调整细微方向（可选）
4. 落地稳住可获得连击加分

构建与运行（根据项目实际技术栈调整）
- 若为 Web 项目：
  - 依赖：Node.js, npm
  - 安装：`npm install`
  - 本地运行：`npm run dev`（或 `npm start`）
  - 打包：`npm run build`
- 若为 Unity / Cocos / Laya 等游戏引擎：
  - 打开工程后按引擎说明导出对应平台（Web/Android/iOS）
  - 可提供导出脚本或 CI 配置以便自动打包
